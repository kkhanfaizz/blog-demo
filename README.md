# Blog Website (Next.js + Tailwind)

📰 A blog website built with Next.js 13 where users can browse multiple blogs.

## ✨ Features
- Blog listing page
- Blog details page
- Navbar and footer
- Deployed on Vercel
- Mobile responsive
